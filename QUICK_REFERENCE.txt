## API QUICK ACCESS

### Dashboard
GET /api/dashboard              → Get metrics
POST /api/field-update          → Update field

### Fields  
GET /api/fields                 → Get all
GET /api/field/<id>             → Get one

### Irrigation
POST /api/irrigation/start      → Start
GET /api/irrigation/history/<id> → History

### Alerts
GET /api/alerts                 → Get all
POST /api/alerts                → Create
PUT /api/alerts/<id>/resolve    → Resolve

### NPK
GET /api/npk-levels/<id>        → Get
POST /api/npk-levels/<id>       → Update

### Yield
GET /api/crop-yield-forecast    → Get forecast

### Weather
GET /api/weather                → Get
POST /api/weather/update        → Update

### Analytics
GET /api/analytics              → Get stats
GET /api/health-check           → Check status

## TROUBLESHOOTING

### Port 5000 in use?
```bash
# Linux/Mac
lsof -i :5000 | grep LISTEN
kill -9 <PID>

# Windows  
netstat -ano | findstr :5000
taskkill /PID <PID> /F
```

### Database issues?
```bash
rm agriculture.db
python app.py
```

### Missing dependencies?
```bash
pip install -r requirements.txt --upgrade
```

### CORS errors?
- Check Flask-CORS is installed
- Clear browser cache (Ctrl+Shift+Del)
- Try incognito mode
