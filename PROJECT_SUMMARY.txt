## Database Structure

### Tables Created (6 tables)
1. **fields** - Field information and current status
2. **npk_levels** - Nutrient level history
3. **irrigation_records** - Irrigation activity logs
4. **crop_yield_forecast** - Yield predictions
5. **alerts** - System alerts and notifications
6. **weather** - Weather forecast data

### Data Relationships
- Fields have many NPK levels
- Fields have many irrigation records
- Fields have many yield forecasts
- Fields have many alerts
- All tables linked via field_id foreign key

## Getting Started

### Quick Start (3 steps)
1. Extract all files
2. Run: `python app.py` or `./start.sh` or `start.bat`
3. Open: `http://localhost:5000`

### Installation Steps
1. Install Python 3.8+
2. Run: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Access in browser
